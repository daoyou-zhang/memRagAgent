# 🎨 5 分钟获取漂亮 3D 美女模型

## 🚀 最快方式（推荐）

### 步骤 1：访问 Sketchfab

打开浏览器，访问：
```
https://sketchfab.com/
```

### 步骤 2：搜索模型

在搜索框输入：
```
anime girl rigged free
```

或者：
```
female character realistic free
```

### 步骤 3：筛选

点击左侧筛选：
- ✅ **Downloadable** （可下载）
- ✅ **Free** （免费）
- ✅ **Rigged** （带骨骼）

### 步骤 4：选择模型

- 点击你喜欢的模型
- 查看预览图
- 确保是完整身体的美女模型

### 步骤 5：下载

- 点击 **"Download 3D Model"** 按钮
- 选择 **"glTF"** 格式
- 下载到电脑

### 步骤 6：放置模型

1. 将下载的 `.glb` 文件重命名为 `avatar.glb`
2. 放到这个位置：
   ```
   agentfront\public\models\avatar.glb
   ```

3. 如果 `models` 文件夹不存在，创建它：
   ```
   agentfront\public\models\
   ```

### 步骤 7：刷新浏览器

- 刷新 http://localhost:5175
- 模型会自动加载！

## 📁 完整路径
