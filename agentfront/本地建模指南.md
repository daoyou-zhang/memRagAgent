## 🌟 方案 1：VRoid Studio（最推荐）⭐⭐⭐⭐⭐

### 为什么选择 VRoid Studio？

- ✅ **完全免费**
- ✅ **中文界面**
- ✅ **超级简单**（像捏脸一样）
- ✅ **质量极高**（动漫风格）
- ✅ **自带表情系统**（52 种表情）
- ✅ **自带骨骼和动画**
- ✅ **直接导出 VRM 格式**（可转 GLB）
- ✅ **景甜风格的美女完全可以做出来**

### 详细步骤

#### 1. 下载 VRoid Studio

**官网**：https://vroid.com/studio

**下载地址**：
- Windows：https://vroid.com/studio/download
- 完全免费，无需注册

**安装**：
- 下载后直接安装
- 大小约 500 MB

#### 2. 创建美女角色

**启动 VRoid Studio**

1. **选择模板**
   - 打开软件
   - 选择 "女性" 模板
   - 点击 "创建"

2. **调整面部**（最重要！）
   
   **脸型**：
   - 点击左侧 "脸" 选项卡
   - 调整脸型：选择 "鹅蛋脸" 或 "瓜子脸"
   - 下巴：稍微尖一点
   - 脸颊：柔和圆润
   
   **眼睛**（景甜风格的关键）：
   - 眼睛大小：调大（70-80%）
   - 眼睛形状：选择 "杏仁眼" 或 "桃花眼"
   - 眼距：适中偏宽
   - 眼睛颜色：深棕色或黑色
   - 睫毛：加长加密
   - 眼影：淡粉色或裸色
   
   **鼻子**：
   - 鼻梁：挺直
   - 鼻头：小巧
   - 鼻翼：不要太宽
   
   **嘴巴**：
   - 嘴唇：饱满
   - 嘴型：微笑唇
   - 颜色：粉红色或玫瑰色
   
   **眉毛**：
   - 形状：柳叶眉
   - 颜色：深棕色
   - 粗细：适中

3. **调整发型**
   
   **长发造型**（景甜风格）：
   - 点击 "头发" 选项卡
   - 选择 "长直发" 或 "波浪长发"
   - 长度：到腰部
   - 刘海：空气刘海或中分
   - 颜色：黑色或深棕色
   - 添加光泽

4. **调整身材**
   
   - 点击 "身体" 选项卡
   - 身高：165-170cm
   - 身材：匀称
   - 胸围：适中
   - 腰围：纤细
   - 臀围：适中

5. **选择服装**
   
   **优雅风格**：
   - 点击 "服装" 选项卡
   - 上衣：白色衬衫或连衣裙
   - 下装：裙子或裤子
   - 颜色：白色、粉色、浅蓝色
   - 添加配饰（可选）

6. **调整表情**
   
   - VRoid Studio 自带 **52 种表情**
   - 包括：微笑、大笑、惊讶、思考、害羞等
   - 这些表情会自动包含在导出的模型中

#### 3. 导出模型

1. **导出 VRM 格式**
   - 点击右上角 "导出"
   - 选择 "VRM 0.0" 或 "VRM 1.0"
   - 设置名称：`my_avatar.vrm`
   - 点击 "导出"

2. **转换为 GLB 格式**
   
   **方法 A：在线转换**（推荐）
   ```
   https://vrm.dev/vrm_converter/
   ```
   - 上传 .vrm 文件
   - 选择 "GLB" 格式
   - 下载转换后的文件
   
   **方法 B：使用 Blender**
   - 安装 Blender：https://www.blender.org/
   - 安装 VRM 插件：https://github.com/saturday06/VRM-Addon-for-Blender
   - 导入 VRM → 导出 GLB

3. **放置模型**
   ```
   agentfront\public\models\avatar.glb
   ```

#### 4. 表情控制

VRoid 模型自带的表情可以通过 **Blend Shapes** 控制：

- `Joy` - 开心
- `Angry` - 生气
- `Sorrow` - 悲伤
- `Fun` - 有趣
- `Surprised` - 惊讶
- `Blink` - 眨眼
- 等等...

## 🎭 方案 2：Ready Player Me（最快）⭐⭐⭐⭐

### 优点
- ✅ 10 分钟完成
- ✅ 在线创建，无需安装
- ✅ 写实风格
- ✅ 直接导出 GLB

### 步骤

1. **访问 Ready Player Me**
   ```
   https://readyplayer.me/
   ```
   
   ⚠️ **注意**：在中国可能需要 VPN

2. **创建角色**
   - 点击 "Create Avatar"
   - 选择 "Full Body"
   - 上传照片（可选）或手动调整
   - 调整面部特征
   - 选择发型和服装

3. **导出**
   - 点击 "Download"
   - 选择 "GLB" 格式
   - 下载到本地

4. **放置**
   ```
   agentfront\public\models\avatar.glb
   ```

## 🔧 方案 4：Blender（完全自定义）⭐⭐⭐

### 适合人群
- 有 3D 建模经验
- 想要完全自定义
- 有充足时间

### 步骤

1. **下载 Blender**
   ```
   https://www.blender.org/
   ```

2. **使用插件**
   
   **MB-Lab**（人体生成插件）：
   ```
   https://github.com/animate1978/MB-Lab
   ```
   - 自动生成人体
   - 调整面部和身材
   - 添加材质和纹理

3. **添加表情**
   - 使用 Shape Keys
   - 手动创建表情

4. **导出 GLB**
   - File → Export → glTF 2.0 (.glb)
