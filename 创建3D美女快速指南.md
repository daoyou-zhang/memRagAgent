# 🎨 创建漂亮 3D 美女 - 30 分钟快速指南

## 🎯 目标

30 分钟内创建一个漂亮的、带表情的 3D 美女模型（景甜风格）

---

## ⚡ 快速步骤

### 第 1 步：下载 VRoid Studio（5 分钟）

1. **访问官网**
   ```
   https://vroid.com/studio
   ```

2. **下载 Windows 版本**
   - 点击 "Download"
   - 选择 Windows 版本
   - 大小约 500 MB

3. **安装**
   - 双击安装包
   - 一路 "下一步"
   - 完成安装

---

### 第 2 步：创建美女角色（20 分钟）

#### 2.1 启动并选择模板（1 分钟）

1. 打开 VRoid Studio
2. 点击 "新建"
3. 选择 "女性" 模板
4. 点击 "创建"

#### 2.2 调整面部（10 分钟）⭐ 最重要！

**脸型**（2 分钟）：
- 左侧点击 "脸" 图标
- 脸型：选择 "鹅蛋脸"
- 下巴：调尖一点（70%）
- 脸颊：柔和（60%）

**眼睛**（3 分钟）：
- 大小：**调大到 75-80%**（景甜风格的关键！）
- 形状：选择 "杏仁眼"
- 眼距：稍微宽一点（55%）
- 颜色：深棕色或黑色
- 睫毛：长度 80%，密度 70%
- 眼影：淡粉色

**鼻子**（1 分钟）：
- 鼻梁：挺直（70%）
- 鼻头：小巧（40%）
- 鼻翼：窄（45%）

**嘴巴**（2 分钟）：
- 嘴唇：饱满（65%）
- 嘴型：微笑唇
- 颜色：粉红色（#FF6B9D）

**眉毛**（1 分钟）：
- 形状：柳叶眉
- 颜色：深棕色
- 粗细：适中（50%）

**皮肤**（1 分钟）：
- 肤色：白皙（#FFE4D6）
- 光泽：适中（60%）

#### 2.3 设置发型（5 分钟）

1. 点击 "头发" 图标

**发型选择**：
- 样式：长直发 或 波浪长发
- 长度：到腰部（80%）

**刘海**：
- 样式：空气刘海 或 中分
- 长度：到眉毛

**颜色**：
- 主色：黑色 (#1A1A1A) 或 深棕色 (#2C1810)
- 高光：深棕色（可选）

**质感**：
- 光泽：70%
- 柔顺度：80%

#### 2.4 调整身材（2 分钟）

1. 点击 "身体" 图标

**身高**：
- 165-170cm（调整滑块到 60-70%）

**身材比例**：
- 头身比：7.5 头身
- 肩宽：适中（50%）
- 胸围：适中（55%）
- 腰围：纤细（40%）
- 臀围：适中（55%）
- 腿长：修长（65%）

#### 2.5 选择服装（2 分钟）

1. 点击 "服装" 图标

**推荐搭配**：

**方案 A：优雅连衣裙**
- 上衣：白色连衣裙
- 颜色：纯白色 (#FFFFFF)
- 长度：及膝

**方案 B：休闲风**
- 上衣：白色衬衫
- 下装：牛仔裤或短裙
- 颜色：白色 + 蓝色

**方案 C：可爱风**
- 上衣：粉色 T 恤
- 下装：白色短裙
- 颜色：粉色 + 白色

**鞋子**：
- 白色运动鞋 或 小皮鞋

---

### 第 3 步：导出模型（3 分钟）

#### 3.1 导出 VRM（1 分钟）

1. 点击右上角 "导出" 按钮
2. 选择 "VRM 0.0" 格式
3. 设置名称：`my_beautiful_avatar`
4. 点击 "导出"
5. 选择保存位置（桌面）

#### 3.2 转换为 GLB（2 分钟）

**方法 A：在线转换**（推荐）

1. 访问转换网站：
   ```
   https://vrm.dev/vrm_converter/
   ```

2. 上传刚才导出的 `.vrm` 文件

3. 选择输出格式：`GLB`

4. 点击 "Convert"

5. 下载转换后的 `.glb` 文件

6. 重命名为：`avatar.glb`

---

### 第 4 步：放置并使用（2 分钟）

#### 4.1 放置模型

1. 打开项目文件夹：
   ```
   D:\workspace\memRagAgent\agentfront\public\models\
   ```

2. 将 `avatar.glb` 文件放到这个文件夹

3. 确保文件名是：`avatar.glb`

#### 4.2 刷新浏览器

1. 打开浏览器：
   ```
   http://localhost:5175
   ```

2. 按 `F5` 刷新页面

3. 等待 3-5 秒加载

4. **完成！** 🎉

## 💡 常见问题

### Q: VRoid Studio 是免费的吗？
A: 是的，完全免费！

### Q: 需要注册账号吗？
A: 不需要，直接下载使用。

### Q: 转换网站安全吗？
A: 是的，vrm.dev 是官方推荐的转换工具。

### Q: 模型文件多大？
A: 通常 5-15 MB，取决于细节程度。

### Q: 可以修改已创建的模型吗？
A: 可以！保存 VRoid 项目文件（.vroid），随时可以重新编辑。

### Q: 如果模型不显示怎么办？
A: 检查：
1. 文件名是否为 `avatar.glb`
2. 文件是否在 `public/models/` 文件夹
3. 浏览器控制台是否有错误
4. 刷新浏览器（F5）
